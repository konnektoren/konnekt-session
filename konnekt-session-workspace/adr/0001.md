# ADR-0001: Use Rust for Implementation

**Status**: Accepted | **Date**: 2025-12-29

## Context

Need a language with strong type safety, WASM compilation, Ed25519 crypto, memory safety without GC, and cross-platform support (browser + native).

## Decision

Use **Rust** as the primary implementation language.

## Why

- Type system encodes domain invariants at compile time (DDD alignment)
- First-class WASM via `wasm-bindgen`, `wasm-pack`, `web-sys`
- `ed25519-dalek` — audited, pure Rust, WASM-compatible
- Zero-cost abstractions, no GC pauses
- Single codebase targets both WASM and native

## Trade-offs

- ✅ Compile-time bug detection, excellent performance
- ❌ Steeper learning curve (ownership/borrowing)
- ❌ Slower compile times, smaller talent pool
