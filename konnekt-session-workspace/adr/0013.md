# ADR-0013: Use instant for Cross-Platform Time

**Status**: Accepted | **Date**: 2025-12-29

## Context

`std::time::Instant` panics at runtime in WASM. Need monotonic time for heartbeat timeouts, host delegation timer (30s), performance measurement.

## Decision

Use the **`instant`** crate — drop-in replacement for `std::time::Instant`.

```toml
instant = { version = "0.1", features = ["wasm-bindgen"] }
```

```rust
use instant::{Instant, Duration};

let start = Instant::now();
if start.elapsed() > Duration::from_secs(30) {
    // host delegation timeout
}
```

## Why

- Same API as `std::time::Instant` — minimal code changes
- Uses `performance.now()` in browser (WASM-native)
- ~2KB WASM overhead
- Already a transitive dependency via Matchbox and other crates

## Trade-offs

- ✅ Cross-platform, zero API change, minimal overhead
- ❌ One more explicit dependency (already transitive)
